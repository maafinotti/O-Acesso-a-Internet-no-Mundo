O Nosso Projeto

O Acesso à Internet No Mundo


       O processo para realização desse projeto se iniciou quando criamos um arquivo no Google Docs com o intuito de nos organizarmos e colocarmos todas as nossas ideias do que seria legal e o que agregaria para o trabalho. Dessa forma, cada um contribuiu com ideias, pesquisas, links e portais, assim ficou mais fácil visualizar nosso ponto de partida.
       Procurar Datasets que apresentavam informações pertinentes foi o primeiro desafio, pois há muitos datasets com conteúdos irrelevantes ou inúteis para o que nós queríamos apresentar no nosso projeto.
       Após muitas pesquisas, sites analisados e debates entre o grupo, selecionamos 5 Datasets (disponibilizados em Fontes). Porém, para começar a fazer os gráficos foi preciso juntar os 5 arquivos e depois tratar para que ficassem apenas informações importantes para execução dos gráficos.
       E esse foi o nosso maior desafio, precisamos excluir dados desnecessários, transformar dados object para int ou float, para que fosse possível fazer os gráficos e análises estatísticas.
       O próximo passo foram os gráficos, decidimos os critérios e as variáveis para produção desses, como “População por país” e “Usuários de internet (%) por país”, de acordo com os dados que o nosso Dataset tratado apresentava. Depois escolhemos qual seria a formatação do gráfico (linha, barra, dispersão). Utilizamos Python para realização desses e bibliotecas como Matplotlib e Seaborn (visualização dos códigos no link do GitHub).
       Para o mapa coroplético, foi necessário uma atenção a mais, pois ao juntar os datasets com o arquivo em Geojson, que contém a geometria dos países, alguns foram excluídos por apresentarem um nome diferente (um acento ou forma de escrever) e, assim, nosso mapa ficou falhado, com alguns espaços em branco. Dessa forma, precisamos adicionar alguns dados manualmente no nosso dataset, principalmente aqueles que ficam mais visíveis no mapa, um exemplo foi a Rússia.
       Quando todos os gráficos ficaram prontos, fizemos uma análise de cada um. Criamos insights e relacionamos com informações econômicas e sociais dos países para dar credibilidade para o nosso projeto.
       Quando esses insights foram finalizados, começamos a preparar nosso site, fizemos o design no Figma e utilizamos o framework Flask para linkar o HTML e CSS e criar nosso site “Internet Access In The World”.
       E por fim, colocamos os arquivos no GitHub e organizamos os últimos detalhes como design do site, editar fontes, cores.
